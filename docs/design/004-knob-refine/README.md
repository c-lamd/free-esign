# Sketch 004: Knob Refine (the mix)

Synthesis sketch. In 003 you chose the **knob** (over stepper/fader) but found the rebuilt one too
maximalist — you preferred the simpler 002 version. This takes **002's silhouette** (tick ring +
pointer, moderate shadow), keeps it **fully functional** (drag ↕, document scales), and varies only
*how much position indication* it carries.

## Design Question
How much indication does the calm knob need to still read as operable — without tipping back into
"synth module"?

## Candidates (purest → most indication)
- **A — Pure** — pointer + tick ring only. The cleanest, basically 002, now wired.
- **B — Pure + marker** ⭐ *(the mix)* — adds a single accent dot that rides the tick ring with the
  value. Indicative and obviously "set to a position," still calm. The best of 002 + 003.
- **C — Pure + hairline arc** — a thin 2px accent arc (003's sweep dialed way back). Slightly more
  signal than B; closest to 003 without the bulk.

## What to Look For
- Does **B**'s ring marker give you the "I can see where it's set" read that the bare pointer (A) lacks?
- Is **C**'s thin arc still too much, or a nice subtle gauge?
- Shadow weight: this is intentionally between 002 (too flat) and 003 (too heavy) — does it sit right?

## Decision
**C — Pure + hairline arc** (chosen). The thin accent arc gives a clean gauge read without the bulk.
Refinement applied per feedback: the **tick ring is bounded to the knob's 270° travel** (blank at the
bottom dead-zone) so ticks + arc + pointer all share the same sweep, like a real instrument. Closes
the zoom-control question.
